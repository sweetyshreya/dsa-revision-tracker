# DSA Spaced Repetition Tracker

A web application that helps students revise Data Structures and Algorithms problems using spaced repetition.

## Features

- Add/Edit/Delete problems
- Track revision schedule
- Smart spaced repetition
- Monthly review mode
- Undo last revision
- LocalStorage support
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript

## Future Improvements

- React
- Node.js
- Authentication
- Database
